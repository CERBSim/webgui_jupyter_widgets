
webgui_jupyter_widgets
=====================================

Version: |release|

Jupyter widgetds library for webgui js visualization library'


Quickstart
----------

To get started with webgui_jupyter_widgets, install with pip::

    pip install webgui_jupyter_widgets

or with conda::

    conda install webgui_jupyter_widgets
